# 坦克大战 - 游戏实现

## 技术选择说明

### 为什么选择 HTML5 Canvas + JavaScript？

1. **跨平台兼容性** - 任何现代浏览器都能运行，无需安装任何依赖
2. **部署方便** - 只需上传HTML和JS文件，即可在任何web服务器上运行
3. **开发效率** - JavaScript开发速度快，调试便利
4. **代码可读性** - 采用模块化设计，结构清晰易于维护
5. **性能** - Canvas 2D渲染性能足以支撑这类游戏

## 项目结构

```

│   ├── Game.js             # 游戏引擎核心
│   └── main.js             # UI交互和程序入口
├── 游戏策划文档.md         # 完整的游戏设计文档
└── README.md               # 本文件
```

## 游戏操作说明

### 单人模式
- **WASD** - 坦克移动
- **← →** (左右方向键) - 炮塔转向
- **空格** - 发射子弹
- **P** - 暂停/继续

**玩家1：**
- WASD - 移动

**玩家2：**

- J/L - 转向（左/右）
- 回车 - 射击

## 代码架构

### 采用模块化设计原则

每个类都有明确的职责：

1. **Vector2** - 向量运算辅助类
2. **Bullet** - 管理子弹生命周期和碰撞检测
3. **Powerup** - 处理道具的效果和显示
4. **Tank** - 坦克行为、射击、AI控制
5. **MapGenerator** - 随机地图生成和障碍物管理
6. **GameState** - 游戏状态、分数、关卡进度
7. **Game** - 游戏引擎核心（更新循环、碰撞检测、渲染）
8. **main.js** - UI交互和初始化

### 优势

- **易于维护** - 每个类职责单一，易于定位问题
- **可扩展性** - 添加新的坦克类型、道具、敌人只需修改相应类
- **代码复用** - 工具函数库(utils.js)避免重复代码
- **清晰的数据流** - 游戏状态集中管理，不会出现状态混乱

## 性能优化

1. **Canvas优化**
   - 只重绘变化的对象，不重绘静态背景
   - 使用RequestAnimationFrame实现平滑60FPS

2. **对象池**
   - 子弹和道具的数量有限制，避免内存溢出

3. **碰撞检测优化**
   - 矩形碰撞检测（O(n)复杂度）
   - 只检测活跃对象

4. **AI优化**
   - 敌坦克AI决策每2秒更新一次，不是每帧

## 自定义和扩展

### 修改游戏难度
编辑 `js/config.js` 中的 `CONFIG.DIFFICULTY_MULTIPLIERS` 对象。

### 添加新关卡
编辑 `js/config.js` 中的 `CONFIG.LEVELS` 数组，按照现有关卡格式添加新关卡。

### 修改坦克属性
编辑 `CONFIG.ENEMY_TANKS` 配置对象，调整敌坦克的生命值、速度等属性。

### 修改道具参数
编辑 `CONFIG.POWERUPS` 配置对象，调整道具的持续时间、效果倍数等。

### 示例：增加敌坦克难度
```javascript
// 在 config.js 中修改
normal: {
    HEALTH: 30,  // 原来25，现改为30
    SPEED: 140,  // 原来120，现改为140
    FIRE_RATE: 1.0,  // 原来0.8
    // ...
}
```

## 常见问题

### Q: 游戏运行很卡？
A: 这可能是：
1. 浏览器性能问题 - 尝试用Chrome或Firefox
2. 背景运行程序占用资源 - 关闭其他程序
3. 关卡敌坦克过多 - 降低难度重新开始

### Q: 想修改游戏参数但不知道从何开始？
A: 所有游戏参数集中在 `js/config.js` 文件中，每个参数都有清晰注释，可以直接修改数值。

### Q: 如何添加新的道具类型？
A: 
1. 在 `js/types.js` 中添加新的 `POWERUP_TYPE`
2. 在 `js/config.js` 中添加配置信息
3. 在 `js/Powerup.js` 的 `getEffectInfo()` 中添加描述
4. 在 `js/Game.js` 的 `onEnemyTankDefeated()` 中添加生成几率

### Q: 游戏可以联网对战吗？
A: 当前版本暂不支持，但可以通过添加WebSocket或WebRTC连接实现联网功能。
